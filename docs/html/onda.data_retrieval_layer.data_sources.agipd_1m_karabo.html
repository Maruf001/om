The agipd_1m_karabo Module¶

Retrieval of AGIPD 1M detector data from Karabo.

This module contains functions that retrieve data from an AGIPD 1M x-ray detector using the Karabo framework.

get_peakfinder8_info()[source]¶

Retrieves the peakfinder8 information for the AGIPD 1M detector.

Returns

a named tuple storing the peakfinder8 information.

Return type

Dict[str, Union[int, float]]

get_num_frames_in_event(event)[source]¶

Gets the number of frames in an AGIPD 1M event retrieved from Karabo.

This function retrieves the number of frames in each event for the detector identified by the ‘karabo_detector_label’ entry in the ‘DataRetrievalLayer’ configuration parameter group.

NOTE: This function is designed to be injected as a member function into an DataEvent object.

Parameters

event (DataEvent) – an object storing the event data.

Returns

the number of frames in the event.

Return type

int

detector_data(event)[source]¶

Retrieves from Karabo one frame of AGIPD 1M detector data.

This function retrieves a data frame from the detector identified by the ‘karabo_detector_label’ entry in the ‘DataRetrievalLayer’ configuration parameter group.

Parameters

event (DataEvent) – an object storing the event data.

Returns

one frame of detector data.

Return type

numpy.ndarray

detector_gain(event)[source]¶

Retrieves from Karabo gain status data for one frame of AGIPD 1M.

This function retrieves the gain status information for a data frame from the detector identified by the ‘karabo_detector_label’ entry in the ‘DataRetrievalLayer’ configuration parameter group.

Parameters

event (DataEvent) – an object storing the event data.

Returns

one frame of detector data.

Return type

numpy.ndarray
