The data_handlers_hidra Module¶

Retrieval and handling of data events from HiDRA.

This module contains DataEventHandlers for events retrieved from the HiDRA framework.

class P11Petra3DataEventHandler(monitor_parameters, source)[source]¶

Bases: om.data_retrieval_layer.base.OmDataEventHandler

DataEventHandler for events retrieved from HiDRA at P11 (PETRA III).

See documentation of the corresponding function in the base class. This class handles Pilatus detector data events retrieved from HiDRA at the P11 beamline of the PETRA III facility.

Parameters

  • monitor_parameters (MonitorParams) – An object storing the OM monitor parameters from the configuration file.

  • source (str) – A string describing the data source.

property data_extraction_funcs¶

Retrieves the Data Extraction Functions for HiDRA events at P11.

See documentation of the corresponding function in the base class.

Return type

Dict[str, Callable[[Dict[str, Dict[str, Any]]], Any]]

Returns

A dictionary storing the implementations of the Data Extraction Functions available to the current DataEventHandler.

  • Each dictionary key defines the name of a function.

  • The corresponding dictionary value stores the function implementation.

initialize_event_handling_on_collecting_node(node_rank, node_pool_size)[source]¶

Initializes P11 HiDRA event handling on the collecting node.

See documentation of the corresponding function in the base class. This function announces OM to HiDRA at P11 and configures HiDRA to send data event to the processing nodes.

Parameters

  • node_rank (int) – The rank, in the OM pool, of the processing node calling the function.

  • node_pool_size (int) – The total number of nodes in the OM pool, including all the processing nodes and the collecting node.

Return type

Any

Returns

An optional initialization token.

Raises

OmHidraAPIError – if the initial connection to HiDRA fails.

event_generator(node_rank, node_pool_size)[source]¶

Retrieves HiDRA events at P11.

See documentation of the corresponding function in the base class. This function retrieves events from HiDRA at P11. Each HiDRA event contains data from a single CBF file written by a Pilatus detector.

Parameters

  • node_rank (int) – The rank, in the OM pool, of the processing node calling the function.

  • node_pool_size (int) – The total number of nodes in the OM pool, including all the processing nodes and the collecting node.

Yields

A dictionary storing the event data.

Raises

OmHidraAPIError – if the initial connection to HiDRA fails.

Return type

Generator[Dict[str, Any], None, None]

open_event(event)[source]¶

Opens a P11 HiDRA event.

See documentation of the corresponding function in the base class. At P11, each HiDRA event contains data from a single Pilatus CBF data file. This function opens the file and associates its binary content to the ‘data’ key of the ‘event’ dictionary.

Parameters

event (Dict[str, Any]) – A dictionary storing the event data.

Return type

None

close_event(event)[source]¶

Closes a P11 HiDRA data event.

See documentation of the corresponding function in the base class. At P11, each HiDRA event contains data from a single Pilatus CBF data file. The file does not need to be closed, therefore this function does nothing,

get_num_frames_in_event(event)[source]¶

Gets the number of frames in a P11 HiDRA data event.

See documentation of the corresponding function in the base class. At P11, each HiDRA event contains data from a single Pilatus CBF data file. The file usually contains just one frame, therefore this function always returns 1.

Parameters

event (Dict[str, Any]) – A dictionary storing the event data.
