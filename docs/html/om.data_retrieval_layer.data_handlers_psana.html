The data_handlers_psana Module¶

Retrieval and handling of data events from psana.

This module contains classes that retrieve and process data events from the psana framework.

get_num_frames_in_event(event)[source]¶

Gets the number of frames in an event retrieved from psana at the LCLS facility.

See documentation of the function in the base class: get_num_frames_in_event() .

Psana events are frame-based, and always contain just one frame. This function always returns 1.

Returns

the number of frames in the event.

int
